# 太陽系縮尺シミュレーター

MapLibre GL JSを使用した太陽系の縮尺を地図上で可視化するWebアプリケーションです。

## 概要

地球の直径を指定することで、太陽系の惑星軌道を同じ縮尺で地図上に表示します。実際の距離感を視覚的に理解できるツールです。

## 機能

### 縮尺設定
- **地球の直径を指定**: 任意の大きさ（cm単位）で地球を設定
- **太陽の直径を自動計算**: 地球のサイズに応じて太陽の大きさも連動
- **双方向入力**: 地球または太陽のどちらからでも計算可能

### 惑星表示
- 8つの惑星の軌道を円で表示
- 各惑星を色分けして表示
- 惑星本体も縮尺に合わせて表示
- 惑星名のラベル表示

### 地図機能
- **背景地図の切り替え**:
  - OpenStreetMap（標準）
  - 地理院地図（標準）
  - 地理院地図（淡色）
  - 地理院地図（白地図）
  - 地理院地図（衛星写真）
- **太陽の位置を変更可能**: ドラッグ操作で自由に移動

## 使い方

1. ブラウザで `solar-system-map.html` を開く
2. 地球の直径を入力（デフォルト: 1.3cm）
3. 「描画更新」ボタンをクリック
4. 太陽マーカーをドラッグして位置を変更可能
5. 背景地図をドロップダウンから選択

## 初期設定

- **初期中心座標**: 和歌山県 四季の郷公園 (135.2597718, 34.2161407)
- **初期ズームレベル**: 14
- **デフォルト地球直径**: 1.3 cm

## 動作例

地球の直径を1.3cmに設定した場合:
- 太陽の直径: 約1.42m
- 地球の軌道半径(1AU): 約1,510m
- 海王星の軌道: 約45km
