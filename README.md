# 宇宙スケール体感シミュレーター集

MapLibre GL JSとChart.jsを使用した、宇宙のスケールを視覚化するWebアプリケーション集です。

## 概要

太陽系から銀河系外まで、さまざまな宇宙のスケールを地図上や金額に換算して体感できる5つのシミュレーターを提供します。実際の距離感やサイズ感を視覚的に理解できるインタラクティブなツール集です。

## シミュレーター一覧

### 1. 太陽系大きさ体感マップ (`solar-system-map.html`)
地球の直径を指定することで、太陽系の惑星軌道を同じ縮尺で地図上に表示します。
- **対象**: 太陽系内の8惑星
- **縮尺設定**: 地球の直径（cm単位）
- **表示内容**: 惑星軌道、惑星本体、太陽

### 2. 恒星間距離体感マップ (`stellar-distance-map.html`)
太陽から近傍の恒星までの距離を地図上に縮尺して表示します。
- **対象**: プロキシマ・ケンタウリ、シリウス、ベガなど主要恒星
- **縮尺設定**: 1光年を何kmで表示するか
- **表示内容**: 恒星位置、距離（光年）、星座情報

### 3. 恒星比較シミュレーター (`stellar-comparison-map.html`)
太陽と他の恒星の大きさを比較し、地図上で視覚化します。
- **対象**: ベテルギウス、リゲル、アンタレスなど巨大恒星
- **縮尺設定**: 太陽の直径（メートル単位）
- **表示内容**: 恒星の相対的サイズ、分類、実直径

### 4. 銀河系距離シミュレーター (`galaxy-distance-map.html`)
銀河系の大きさと近傍銀河までの距離を地図上に表示します。
- **対象**: 銀河系、アンドロメダ銀河、大マゼラン雲など
- **縮尺設定**: 銀河系の直径を何kmで表示するか
- **表示内容**: 銀河位置、距離（光年）、銀河の種類

### 5. 宇宙距離金額換算シミュレーター (`space-money-simulator.html`)
宇宙の距離を金額に換算し、グラフとビジュアルで表現します。
- **換算モード**: 「1km = 1円」または「1光年 = 1円」
- **対象**: 地球周辺から銀河系外まで
- **表示内容**: 金額換算グラフ、比較表、視覚的表現

## 共通機能

### 地図機能（地図ベースシミュレーター共通）
- **背景地図の切り替え**:
  - OpenStreetMap（標準）
  - 地理院地図（標準）
  - 地理院地図（淡色）
  - 地理院地図（白地図）
  - 地理院地図（衛星写真）
- **中心点のドラッグ移動**: 太陽や銀河系中心を自由に移動可能
- **インタラクティブな描画更新**: 縮尺変更や位置移動に応じてリアルタイム更新
- **ズーム・パン操作**: 地図の拡大縮小と移動

### UI要素
- **コントロールパネル**: 縮尺設定と描画更新ボタン
- **凡例パネル**: 表示オブジェクトのリストと色分け
- **範囲外パネル**: 地図の表示範囲外のオブジェクト情報
- **全画面モード**: F11キーで全画面表示
- **パネル折りたたみ機能**: トグルボタンで各パネルの表示/非表示切り替え

## 使い方

### 基本操作（地図ベースシミュレーター共通）
1. ブラウザで対象のHTMLファイルを開く
2. コントロールパネルで縮尺を設定
3. 「描画更新」ボタンをクリックして描画
4. 中心マーカーをドラッグして位置を変更
5. 背景地図をドロップダウンから選択
6. 地図をクリック・ドラッグして閲覧

### 金額換算シミュレーター
1. ブラウザで `space-money-simulator.html` を開く
2. 「kmモード」または「光年モード」を選択
3. グラフとビジュアル表現で金額を比較
4. スクロールで全データを閲覧

## 開発背景

宇宙のスケール感は、数字だけでは実感しにくいものです。このプロジェクトは、地図や金額という身近な概念を使って、太陽系から銀河系外までの膨大な距離やサイズを直感的に理解できるようにすることを目的としています。

各シミュレーターは独立して動作し、それぞれ異なるスケールの宇宙を体感できます:
- **太陽系**: 惑星間の距離感
- **恒星間**: 恒星までの途方もない距離
- **恒星比較**: 恒星の巨大なサイズ
- **銀河系**: 銀河の大きさと銀河間の距離
- **金額換算**: 距離を金額で直感的に理解
